MAPSDUMPER

  • Demo
  • Installation
  • Feature
  • Credits

A tool for dumping place details from Google Maps such as phone numbers, emails, websites, and reviews

##Installation

  • > git clone https//:github.com/tegal1337/mapsdumper
  • > cd mapsdumper
  • > npm install
  • > node index.js
